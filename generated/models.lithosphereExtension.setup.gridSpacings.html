models.lithosphereExtension.setup.gridSpacings¶

models.lithosphereExtension.setup.gridSpacings(params, xsize, ysize, grid, t_curr)¶

Calculates the new grid point spacings based on the current xsize and ysize. This version contructs a non-uniform grid with a central-upper high resolution region and decreasing resolution outward from this.

Parameters¶

paramsParameters Class

Parameters object containing all simulation parameters for the system.

xsizeFLOAT

Physical size of the system in x-direction.

ysizeFLOAT

Physical size of the system in y-direction.

gridOBJ

The grid object into which the new node positions will be written.

t_currFLOAT

The current simulation time, to determine whether to set up grid from scratch or extend an existing one.

Returns¶

None.
