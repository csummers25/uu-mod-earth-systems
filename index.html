Modelling Earth Systems - Project code documentation¶

A python reworking of the thermomechanical code developed in the book Numerical Geodynamic Modelling by Taras Gerya [1] . It uses a finite difference staggered grid approach to solve the Stoke’s, continuity and temperature equations, along with markers for advection.

[1]

Gerya T. MATLAB program examples. In: Introduction to Numerical Geodynamic Modelling. Cambridge University Press; 2019:425-437. DOI
